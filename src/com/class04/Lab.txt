-------------------------------------------------------------------------------------------------------------
                                                Important
A program, also known as software, is a set of instructions that a computer follows to perform a task.
Before you start coding always think through the design of what it is that you're coding. 
-------------------------------------------------------------------------------------------------------------
Given a problem, you can solve that problem using an algorithm, which is a high-level step-by-step 
guide to complete a task. Generally, it involves 5 steps:
--------------------------------------------------------------------------------------------------------------
Step 1 - Declare known values & unknown values
Step 2 - Initialize known values
Step 3 - Get values for the unknown
Step 4 – Implement your code logic
Step 5 – Output the program

                                               Practice Program 1
                              Topics: Data Types, Variables, Arithmetic Operators
---------------------------------------------------------------------------------------------------------------
Write a program to compute the distance in miles light travels in 100 days, when the approximate speed of light 
is equal to 186000 miles per second. The output of your program should be: 

In 100 days, light will travel about 16070400000000 miles.

*Hint: speed = distance/time
*Pro Tip: Name your class as: LightDistanceCalculator

                                               Practice Program 2
                             Topics: Data Types, Variables, Arithmetic Operators
---------------------------------------------------------------------------------------------------------------
Write a program to compute the area of a circle with radius 10.8. 
The output of your program should be: 

Area of the circle with radius 10.8 = 366.436224

*Pro Tip: Name your class as: CircleAreaCalculator
---------------------------------------------------------------------------------------------------------------
                                               Practice Program 3
                        Topics: Arithmetic Operators, Modulus Operator, Operator Precedence
---------------------------------------------------------------------------------------------------------------
Write a program to add 12 to the number 2223 and then divide it by 3. Now, the modulus of the quotient is taken 
with 7 and then multiply the resultant value by 5. Display the final result. 
The output of your program should be:

Final Result: 15

*Hint: Create variables for the given literals and consider order of precedence
*Pro Tip: Name your class as: ArithmeticOperators
---------------------------------------------------------------------------------------------------------------
                                               Practice Program 4
                                  Topics: Arithmetic Compound Assignment Operators
---------------------------------------------------------------------------------------------------------------
Write a program to increase the value of an employee’s salary by 18%. The initial salary is $1600.00. 
The output of your program should be: 
Total Increment is: 288.0 
Total Incremented Salary is: 1888.0

*Hint: X% of Y = (X / 100) * Y
*Pro Tip: Name your class as: SalaryIncrementer
---------------------------------------------------------------------------------------------------------------
                                               Practice Program 5
                       Topics: Arithmetic Operators, Assignment Operator, Modulus Operator
---------------------------------------------------------------------------------------------------------------
Write a program to reverse a 3-digit number e.g. 687 to 786. The output of your program should be: 
Reverse: 786

*Hint: Use division and modulus operators to get first, second, and third digits of the 3-digit number
*Pro Tip: Name your class as: DigitReverser

                                               Practice Program 6
                          Topics: String Concatenation, Concatenation Operator, Scanner class
---------------------------------------------------------------------------------------------------------------
Write a program to take your first name, middle name, and last name as user inputs. Concatenate your names to 
print your full name on the console.
The sample input and output of your program should be:

Enter your firstname >>>John
Enter your middle name >>>Vilde
Enter your lastname >>>Martin
My full name is: John Vilde Martin

*Pro Tip: Name your class as: StringConcatenator

                                               Practice Program 7
                                              Topics: Scanner class
---------------------------------------------------------------------------------------------------------------
Write a program to calculate an employee’s gross pay. Take pay rate and work hours from user input.
The sample input and output of your program should be:

Enter the number of hours you worked. >>>40
Enter your pay rate. >>>30
Your gross pay is: $1200.0

*Hint: Gross pay = hours worked * hourly rate
*Pro Tip: Name your class as: GrossPayCalculator

                                               Practice Program 8
                                        Topics: If statement, Scanner class
---------------------------------------------------------------------------------------------------------------
All salespeople get a payment of $700 a week. Salespeople who exceed 10 sales get an additional bonus of $300. 
Write a program with extra sales as input from the user and calculate the employee’s total salary in a week.
The sample input and output of your program should be:

How many sales did the employee make this week?
12
The employee's pay is $1000.0

*Pro Tip: Name your class as: SalaryCalculator
---------------------------------------------------------------------------------------------------------------
                                               Practice Program 9
                                      Topics: If-Else statement, Scanner class
---------------------------------------------------------------------------------------------------------------
All salespeople are expected to make at least 10 sales each week. Write a program to calculate quota of the 
employees and for those who made 10 sales each week, print a congratulatory message. For those who don't, print 
how many sales they were short.
The sample input and output of your program should be:

Enter the number of sales you made this week:
9
You did not make your quota. You were 1 sales short

*Hint: salesShort = quota - sales

*Pro Tip: Name your class as: QuotaCalculator
